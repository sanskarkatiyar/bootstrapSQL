The aim of Digital Assignment - 2 is to understand the functionalities and differences between file based system and database system.


The following details of the particular Data set and Languages used:

Data Set: Online Retail Data Set

Programming Language: PHP

DBMS: MySQL (Server doesn't support OracleDB)


I have written the front-end in Bootstrap and have PHP files for functionalities that manipulate the Databases.

Click on the Tabs above to switch between functions.



Important Notes/Instructions

1. Use the LIMIT BY Clause in the View Tab, to not overload the Memory of the server.
2. OracleDB was not supported on the server so I had to work with MySQL.
3. SQL Injection or Form Validation has not been taken care of, so please enter values accordingly.
4. Form Validation is not strict, integrated in HTML.
5. Modification of records has not been implemented.
6. Deletion of tuples has to be done one at a time.
7. The Front-end has been written in Bootstrap; However the responsiveness on Mobile devices has not bee written properly.
8. The data set doesn't have a Primary key since it has been maintained in such a manner.
